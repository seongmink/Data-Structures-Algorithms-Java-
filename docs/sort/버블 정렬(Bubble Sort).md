# 버블 정렬(Bubble Sort)

## 개념

- 서로 인접한 두 원소를 검사하여 정렬하는 알고리즘
- 원소를 두 개씩 교환하면서 마지막 원소까지 오는 모습이 거품이 올라오는 것과 비슷하다고 하여 붙여진 정렬 알고리즘
- 시간 복잡도 : O(n^2)

<br>

## 구현

```java
int[] arr = {54, 2, 22, 43, 24, 77, 37, 32, 64, 99, 53};
bubbleSort(arr);
```

```java
public static void bubbleSort(int[] arr) {
    for(int i = 0; i < arr.length; i++) {
        for(int j = 0; j < arr.length - i - 1; j++) {
            if(arr[j] > arr[j+1]) {
                int temp = arr[j+1];
                arr[j+1] = arr[j];
                arr[j] = temp;
            }
        }
    }
}
```

<br>

## 장점과 단점

- 장점
  - 구현하기 **매우 쉬움**.
- 단점
  - 순서에 맞지 않는 요소를 인접한 요소와 교환함.
  - 교환의 횟수가 많음(하나의 요소가 가장 왼쪽에서 가장 오른쪽까지 이동하기까지 모든 요소와 교환)
  - 일반적으로 교환이 이동보다 더 복잡하기 때문에 버블 정렬은 구현하기 쉽지만 거의 쓰이지 않음.
