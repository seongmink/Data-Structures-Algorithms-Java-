# 삽입 정렬(Insertion Sort)

## 개념

- 배열의 모든 원소를 앞에서부터 차례대로 정렬된 배열 부분과 비교하여 자신의 위치를 찾아 삽입하는 정렬 알고리즘
- 정렬의 시작은 두번째 원소부터 시작하며 본인의 왼쪽으로는 이미 정렬된 상태라고 생각한다.
- 왼쪽의 값이 비교하려는 원소의 값 보다 작을 때 까지 위치를 왼쪽으로 이동시키면 된다.
- 시간 복잡도 : O(n^2)

<br>

## 구현

```java
int[] arr = {54, 2, 22, 43, 24, 77, 37, 32, 64, 99, 53};
insertionSort(arr);
```

```java
public static void insertionSort(int[] arr) {
    for(int i = 1; i < arr.length; i++) {
        for(int j = i; j > 0; j--) {
            if(arr[j-1] > arr[j]) {
                int temp = arr[j-1];
                arr[j-1] = arr[j];
                arr[j] = temp;
            }
        }
    }
}
```

<br>

## 장점과 단점

- 장점
  - 구현하기 쉬움.
  - 대부분 원소가 이미 정렬되어 있는 경우 효율적임.
- 단점
  - 비교적 많은 원소들이 이동한다.
  - 데이터의 수가 많고 데이터의 크기가 클 경우에 적합하지 않다.
